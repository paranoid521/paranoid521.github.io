解决方法如下：

Updating net-ssh to 2.9.3.beta1 (as cremno suggested in a comment above) resolved my problem:

For unix:

get install net-ssh -v 2.9.3.beta1
For windows:

gem install net-ssh -v 2.9.3.beta1
Before that I tried, but it didn’t help:

get install net-ssh
